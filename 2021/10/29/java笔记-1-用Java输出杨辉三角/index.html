思路分析

杨辉三角示例：

假设需要打印的行数是row ，num(i，j)代表第i行第j个数。

通过分析不难得知：

  • 第i行要打印的空格数量为row-i
  • 第i行要打印i个数字，且每行第一个和最后一个（第i个都是1）
  • 当num(i,j)不是1时， num(i,j)=num(i-1,j)+num(i-1,j-1)即该数字等于上一行同一位置数字(都是第对应行第j个数字)和上一行前一个数字之和。
